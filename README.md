## 2x2x2 ルービックキューブ計算電卓：使い方

### 1. 基本操作（90度回転）
コマンドラインに文字を入力して `Enter` を押すと、各面が**時計回りに90度**回転します。
* `U` （上面） / `D` （下面）
* `L` （左面） / `R` （右面）
* `F` （前面） / `B` （奥面）

### 2. 180度回転・逆回転（反時計回り）
* **180度回転**: 操作名に `2` をつけます（例: `U2`, `R2`, `F2`）
* **逆回転（逆元）**: 操作名に `'` をつけます（例: `U'`, `L'`, `B'`）
* ただし、これは基本操作の6つにのみ使えます。

### 3. 連続入力（スペース区切り）
複数の操作をスペースで区切って一度に入力すると、**右から順番に**キューブを回します。
* 入力例：`F R B` （奥を回す ➔ 右を回す ➔ 前を回す）

### 4. 便利コマンド
* `AC` ：キューブの状態と、これまでの操作履歴をすべて初期化（クリア）します。
* `save [名前]` ：現在の状態を操作として好きな名前で保存します（例: `save test1`）。
* `load [名前]` ：手動で「置換 $\sigma$」と「ねじれ $v$」を入力して新しい操作を登録します。
